# DisasterResponseSystem - AI-Powered Emergency Response Management System

DisasterResponseSystem is an intelligent disaster response and emergency resource management platform that helps emergency teams receive, classify, prioritize, and dispatch resources to incidents in real time.

The system automatically categorizes emergency requests, recommends the appropriate response team, allocates the nearest available resource, visualizes incidents on an interactive map, and maintains complete incident history.

---

## 📌 Features

### 🚨 Emergency Incident Management
- Report emergency incidents
- Automatic incident classification
- Priority detection (Critical, High, Medium, Low)
- Live incident dashboard
- Incident status tracking
- Resolve incidents

### 🤖 AI-Based Classification
- Automatic emergency category prediction
- Supports:
  - Fire
  - Medical
  - Rescue
  - Flood
  - Food
  - Water
  - General Emergency

### 🚑 Smart Resource Allocation
- Automatically recommends the correct emergency team
- Finds the nearest available resource
- Calculates:
  - Distance
  - Estimated arrival time (ETA)
- Prevents assigning already busy resources

### 🗺 Live Map Visualization
- Interactive map using Leaflet
- Displays:
  - Emergency locations
  - Resource locations
  - Live dispatch movement
- Real-time incident visualization

### 📊 Dashboard
- Total Incidents
- Critical Incidents
- Medical Requests
- Food Requests
- Water Requests
- Incident Statistics Chart
- Recent Incident Activity

### 📂 Incident History
- Stores resolved incidents
- Search previous incidents
- View incident details
- Resolution timestamps

### 🚒 Resource Management
- View all emergency resources
- Resource status monitoring
- Available / Assigned resources
- Resource type information
